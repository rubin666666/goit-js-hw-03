# GoIT JavaScript Homework 03

Це репозиторій з домашнім завданням №3 з курсу JavaScript в GoIT.

## Структура проекту

- `js/task-1.js` - Завдання 1
- `js/task-2.js` - Завдання 2  
- `js/task-3.js` - Завдання 3
- `index.html` - HTML файл для тестування
- `.prettierrc` - Конфігурація Prettier
- `.gitignore` - Файли, які ігноруються Git

## Як запустити

1. Відкрийте `index.html` у браузері
2. Відкрийте консоль розробника (F12)
3. Перевірте результати виконання завдань
